# Carol-Dayanna-Cruz-Munevar-y-Paula-Andrea-Rey-Reyes
El trabajo presenta tres DFD que describen el proceso para calcular el área de un rectángulo, el promedio de tres notas y las soluciones de una ecuación cuadrática. Estos diagramas muestran de forma lógica y ordenada los pasos necesarios para obtener cada resultado, facilitando la comprensión del procedimiento.
